# Plan 13-01: Jungle Atmosphere Background

## Objective

Add a single shared, presentational `JungleBackground` component that paints a subtle
theme-aware atmospheric layer (vertical gradient + low-opacity edge silhouettes) behind
the entire app shell. Day mode shows a warm sand/jungle gradient with a fern-frond band
along the bottom; night mode shows a deep-teal-to-warm-coral sunset gradient with palm
silhouettes at the horizon. The component is mounted once at the outer flex-row root of
`chat.html` so both the sidebar and the chat column sit over the same atmosphere.
Theme switching is CSS-only via Tailwind's `dark:` variant — no inputs, no services. No
animation in this phase (Phase 16 will add motion using the same SVG structure).
