    - groupedSessions returns an array of { label: string; sessions: ConversationSession[] }
    - "Today" bucket: sessions whose createdAt date matches today's calendar date (same year, month, day)
    - "Yesterday" bucket: sessions whose createdAt date is exactly one calendar day before today
    - "Previous 7 days" bucket: sessions whose createdAt is 2–7 calendar days before today (inclusive)
    - "Older" bucket: sessions whose createdAt is 8+ calendar days before today
    - Buckets with zero sessions are excluded from the returned array
    - Within each bucket, sessions are ordered newest-first (descending createdAt)
    - The existing formatDate(ts) method is unchanged
    - The existing onDeleteClick(event, id) method is unchanged
  </behavior>

  <action>
  Define a DateGroup interface locally in history-panel.ts:

    interface DateGroup { label: string; sessions: ConversationSession[]; }

  Add a get groupedSessions(): DateGroup[] getter. Inside it:

  1. Get today's date with time zeroed: new Date() with hours/minutes/seconds/ms set to 0.
  2. Build four empty arrays: todaySessions, yesterdaySessions, prev7Sessions, olderSessions.
  3. For each session in this.sessions, compute daysDiff: Math.floor((startOfToday.getTime() - startOfSessionDay.getTime()) / 86_400_000) where startOfSessionDay mirrors the same zeroing for the session's createdAt.
  4. Route to bucket: daysDiff === 0 → today, daysDiff === 1 → yesterday, daysDiff >= 2 && daysDiff <= 7 → prev7, daysDiff >= 8 → older.
  5. Sort each bucket descending by createdAt: .sort((a, b) => b.createdAt - a.createdAt).
  6. Build result array in order [Today, Yesterday, Previous 7 days, Older], including only non-empty buckets.
  7. Return result array.

  Do NOT remove the existing formatDate or onDeleteClick methods. Do NOT change any @Input or @Output declarations. No imports need changing — DateGroup is a local interface.
  </action>
