# Plan 12-02: Jungle Palette Re-Map

## Objective

Swap the Soft Studio cream/tan + slate palette for a jungle palette: daytime sunlit
greens with beige/sand neutrals and amber highlights (day), deep teals/blues with
warm sunset coral/orange accents (night). Per PAL-04, this is a **hex-only re-map** —
we keep all `studio-*` token names so no template files need to change. Both
tailwind configs (frontend app + ui lib) must stay in lockstep — they currently
duplicate the same color block, and that mirroring is intentional.
