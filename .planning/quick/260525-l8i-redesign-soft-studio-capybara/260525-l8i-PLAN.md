# Quick Task: Soft Studio + Capybara redesign

## Why

The existing desert palette is low-contrast and the western theme is hard to execute cleanly. PNG mascot has unfixable scaling artifacts at non-integer sizes. User wants a modern, publishable look. Token names (`desert-*`, `cactus-green-*`) will be misleading after the palette change — rename to `studio-*` in the same pass so future maintenance is clean.

## Scope

### 1. Palette swap (Tailwind + CSS vars)

Replace `desert-*` and `cactus-green-*` tokens with `studio-*` tokens in:
- `apps/frontend/tailwind.config.js`
- `libs/ui/tailwind.config.js`
- `apps/frontend/src/styles.scss` (CSS custom properties)

### 2. Mascot

Replace `<img src="/assets/creatures/creature.png" ... />` in `libs/ui/src/lib/message-bubble/message-bubble.html` with an inline SVG capybara. SVG fill reads `currentColor` so it picks up the studio palette in day/night mode automatically via `text-studio-accent dark:text-studio-night-accent`.

### 3. Decoration cleanup

Remove the 11 decorative cactus `<svg>` blocks from `apps/frontend/src/app/chat/chat.html`. Let the new modern shadows carry the visual weight.

### 4. Rename usages across templates

Apply old→new token map (longest-prefix-first to avoid partial matches like `desert-night` matching inside `desert-night-amber`) in every `.html`, `.ts`, `.scss` under `apps/frontend/src/` and `libs/ui/src/`.
